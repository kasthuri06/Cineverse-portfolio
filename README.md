CineVerse is a portfolio website for a film/media company, showcasing projects, services, team members, and company information. It allows visitors to explore films, short films, ads, and music videos while giving the admin full control to manage content.

## About
CineVerse provides end-to-end film production services from concept to screen, ensuring cinematic quality and engaging storytelling. It also highlights the team, company mission, and vision, giving users an immersive experience of the studio's work and culture.

## Features
- Display of films, short films, ads, and music videos with posters and video URLs.
- Admin panel to add, edit, or delete projects, services, and team members.
- About section with Mission, Vision, Story, and Team Members with photos.
- Contact form to receive messages from visitors.
- Responsive design for desktop and mobile.
- Cloud-based storage for media files (posters and team photos).

## Technologies
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Cloud Storage:** Cloudinary
- **Authentication:** JWT for admin actions

## Use Case
- Visitors can explore and learn about the studio's projects and team.
- Admins can easily manage content without modifying code.
- Studio can showcase short films, ads, music videos, and animations in a professional portfolio.
